github操作备忘

1.git安装

=============================

下载并安装Git。

2.设置姓名及邮箱

===================================

3.链接GitHub

======================================

  1. 在用户主目录下，查找.ssh目录。如果有，查看目录下有没有id_rsa和id_rsa.pub这两个文件，如果已经有了，跳过第二步。
  2. 新增SSH Key。

此处的邮箱为GitHub的邮箱。

  1. 在用户主目录里找到.ssh目录，里面有id_rsa和id_rsa.pub两个文件，这两个就是SSH Key的秘钥对，id_rsa是私钥，不能泄露出去，id_rsa.pub是公钥。
    复制id_rsa.pub里的内容，在GitHub的Settings里，会有SSH and GPG keys这一选项，在里面点击New SSH key新建，将复制的内容粘贴进去， Add SSH key即可添加。
  2. 测试链接。

可能会出现GitHub的公钥指纹的验证，输入yes即可。

参考链接：

使用 SSH 连接到 GitHub：https://help.github.com/cn/articles/connecting-to-github-with-ssh

GitHub 的 SSH 密钥指纹：https://help.github.com/cn/articles/githubs-ssh-key-fingerprints

4.TortoiseGit工具

=====================================================

下载安装后，在安装目录下或者开始菜单里找到puttygen.exe打开，点击load加载.ssh目录下的私钥文件id_rsa，之后点击save private key保存成ppk文件。

用TortoiseGit克隆时选择SSH方式克隆，同时勾选Load Putty Key，选择生成的ppk文件，这样就不需要每次登陆输入密码了。
